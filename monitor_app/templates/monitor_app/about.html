About SWF Monitor

This application is a monitoring and information service for the ePIC streaming workflow testbed.

It provides a browser-based dashboard, a REST API, and a WebSocket service for monitoring the state of the testbed components (agents).

Features

  • Real-time status updates of monitored agents.
  • A REST API for programmatic access and control.
  • A WebSocket interface for receiving live updates.
  • User authentication and an admin interface for management.
