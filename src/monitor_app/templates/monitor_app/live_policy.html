Live Stream Policy

Sublevel is the event's declared verbosity class — which humans it reaches (high: everyone, including digests and email; normal: live-page watchers; low: verbose viewers only). It is set in the event's code and changed by changing the event. Live is a separate category — "interesting to some humans, now": each event declares a live default, and the override (stored in SysConfig) is the runtime attention knob that wins when set. The live stream view shows the events whose effective state is live.
